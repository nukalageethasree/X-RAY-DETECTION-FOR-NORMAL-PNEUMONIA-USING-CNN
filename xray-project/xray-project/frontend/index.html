How It Works
  • 📥 Upload a chest X-ray image (PA or AP view). The model accepts any common image format and automatically resizes to 150 × 150 px.
  • 🧠 The 25-layer CNN extracts hierarchical features — edges → textures → anatomical structures — through four convolutional blocks with batch normalization and dropout regularisation.
  • 📊 A sigmoid output neuron returns a probability score. Scores ≥ 0.5 are classified as PNEUMONIA; below 0.5 as NORMAL.
  • ⚡ Analysis completes in under 1 second on CPU. The model file is 7.6 MB and runs locally — images never leave your server.
